# Algorithms

Набор учебных задач и решений по алгоритмам на Java и Python.

## Структура

- `src/` — Java-исходники
- `src_python/` — Python-решения
- `bin/` — скомпилированные `.class` (игнорируется в git)
- `.vscode/` — настройки IDE (игнорируется в git)

## Запуск Java

Скомпилировать и запустить конкретный файл:

```
javac -d bin src/App_5_1_del_n.java
java -cp bin App_5_1_del_n
```

## Запуск Python

```
python src_python/del_elements_equals_n.py
```

## Примечания

- Решения могут менять массивы/структуры на месте — см. комментарии в коде.
